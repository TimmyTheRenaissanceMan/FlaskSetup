# FlaskSetup

This is a basic flask setup to help you to get into the workflow faster.

It includes:
1. Flask server
2. SQlaclchemy database
3. Three relational models
4. A static view
5. Authentication
6. Views with POST requests
7. Basic database manipulations
8. A few lines of jinja2
